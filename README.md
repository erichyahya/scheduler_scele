SCELE Summary Dashboard
A browser extension (Tampermonkey userscript) that enhances the SCELE e-learning platform at Universitas Indonesia. It provides a unified dashboard for viewing assignments, quizzes, announcements, and forum discussions across all enrolled courses.
Features

Assignments & Quizzes: view all deadlines in one place, sorted by date, with submission status checking
Announcements: unified feed of announcements from all selected courses
Forums: learning forum discussions with reply detection
Course picker: select which courses to include, remembers your selection
Fast loading: only checks status for upcoming items, skips past deadlines

## Installation

Install Tampermonkey (or Violentmonkey) in your browser
Create a new userscript
Paste the contents of scele-summary.user.js
Save and visit https://scele.cs.ui.ac.id

## Privacy
This script runs entirely in your browser. No data is sent to any external server. All requests go only to scele.cs.ui.ac.id using your existing login session.
